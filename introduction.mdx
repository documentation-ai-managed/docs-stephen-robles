## Overview

Welcome to the Stephen Robles documentation platform. You get a centralized space to organize, manage, and share all your project documentation efficiently. Whether you document iPhone shortcuts, Apple ecosystem tips, or creative workflows, this platform streamlines your process with intuitive tools and customizable features.

Key benefits include seamless collaboration, version control for docs, and easy publishing to your audience. Use the brand color `#219C50` for consistent styling across your pages.

## Brand Guidelines

<Expandable title="Primary Color Usage (#219C50)" default-open="true">

Use the brand color `#219C50` for headers, buttons, and accents. In CSS:

````css
:root {
  --brand-primary: #219C50;
}

.header {
  background-color: var(--brand-primary);
  color: white;
}
````

Avoid overuse; pair with neutrals for readability. Test contrast ratios for accessibility.
